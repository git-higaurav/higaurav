# Hi Gaurav - Personal Portfolio Website

This is a modern personal portfolio website built with [Next.js](https://nextjs.org), showcasing my professional experience, projects, and skills.

## Features

- Responsive design optimized for all devices
- Dark/Light theme support
- Interactive project showcase
- Contact form integration
- Blog section with MDX support
- SEO optimized
- Fast page loads with Next.js 14 features
